


컬럼 카테고리 를 노드로 만들려면 ERD를 어떤 형태의 csv 파일로 저장해야하는지 알려줄래?


ERD 를 graphDB 로 만들려고 해  내가 준 create table 정보를 기반으로 node, edge, property 를 만들 수 있는 있는 csv 파일(, 형태로 구분되도록) 을 만들어줘.
Csv 파일을 업로드할 수 있는 cypher 쿼리도 부탁해 

[csv 파일 조건 ]
* node_table.csv 
    * table_name, table_description 으로 구성
* node_colum.csv
    * table_name, column_name, primary_key, column_description 으로 구성
    * primary_key 는 노드의 property 로 표현
* edge_table_column.csv
    * table_name, column_name 으로 구성
* edge_column_value.csv
    * column_name, value_name 으로 구성
* edge_foreign_key.csv
    * table_name, column_name 으로 구성 


[특수 조건]
* edge_column_value.csv 는 컬럼형태로 값은 빈 것으로 만들어줘. 
* 일부 create table 만 가져온 거라 reference 연결된 테이블이 create table 에 없을 경우 무시하고 만들어줘 




-- 월 CLM 개인 회원 통합 정보
CREATE TABLE mm_clm_psmb_intg_info (
    mbr_csno PRIMARY KEY TEXT
);
-- TL_CLM_고객_혜택_상세_실적
CREATE TABLE tl_clm_cust_bnft_dtl_rsl (
    mbr_csno  PRIMARY KEY TEXT,
    mcno TEXT,
    card_prdt_cd TEXT,
    pchs_prdt_cd TEXT,
    FOREIGN KEY (card_prdt_cd) REFERENCES ld_card_prdt_clas_ogt(card_prdt_cd),
    FOREIGN KEY (pchs_prdt_cd) REFERENCES pchs_prdt_cd(pchs_prdt_cd)
    FOREIGN KEY (mcno) REFERENCES mm_mrch_info(mcno),
    FOREIGN KEY (mbr_csno) REFERENCES mm_clm_psmb_intg_info(mbr_csno),
);
-- 월 가맹점 정보
CREATE TABLE mm_mrch_info (
    mcno PRIMARY KEY TEXT,
    cuof_ymm TEXT,
    mccb_lcl_cd,
    mccb_mcl_cd,
    mccb_cd,
    FOREIGN KEY (mccb_mcl_cd) REFERENCES mccb_mcl_cd(mccb_mcl_cd),
    FOREIGN KEY (MCCB_lcl_cd) REFERENCES MCCB_lcl_cd(mccb_lcl_cd)
);
