



ERD 를 graphDB 로 만들려고 해  내가 준 create table 정보를 기반으로 node, edge, property 를 만들 수 있는 있는 csv 파일(, 형태로 구분되도록) 을 만들어줘.
csv 파일을 업로드할 수 있는 cypher 쿼리도 부탁해 

[csv 파일 조건 ]
* node_table.csv 
    * table_name, table_description 으로 구성
* node_colum.csv
    * table_name, column_name, primary_key, column_description 으로 구성
    * primary_key 는 노드의 property 로 표현
    * column_description 은 노드의 property 로 표현
* edge_table_column.csv
    * table_name, column_name 으로 구성
* edge_column_value.csv
    * column_name, value_name 으로 구성
* edge_foreign_key.csv
    * table_name, column_name 으로 구성 


[특수 조건]
* edge_column_value.csv 는 컬럼형태로 값은 빈 것으로 만들어줘. 
* 일부 create table 만 가져온 거라 reference 연결된 테이블이 create table 에 없을 경우 무시하고 만들어줘 
* 동일 컬럼에 대해 각 테이블 별로 컬럼을 각각 생성하지 말고,  primary_key 에 해당하는 node 를 중심으로 foreign key 들을 연결할 수 있도록 설계해줘.
